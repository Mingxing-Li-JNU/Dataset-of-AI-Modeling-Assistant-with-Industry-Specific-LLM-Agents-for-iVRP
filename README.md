# Dataset-of-AI-Modeling-Assistant-with-Industry-Specific-LLM-Agents-for-iVRP
The core task of workshop VRP modeling lies in the generation of mathematical models. To accommodate model training, this study designs a dedicated dataset for the workshop VRP domain. The dataset integrates problem-model pairs from classic textbooks and high-quality iVRP journal literature, real-world workshop demand cases, and synthetic samples augmented through constraint rewriting, combination, and other methods. Following manual inspection, screening, and revision of all data entries one by one, a total of 204 high-quality data samples are finally constructed.
